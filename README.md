# Minesweeper

A command-line Minesweeper game

## Running the Game

From within the project's root directory, start the game with `$ python buscaminas.py prueba1.txt`. You will need to install python if you haven't already.

## Examples

### 1. 
#### Input

```
4 4
*...
....
.*..
....
```

#### Output

```
*100

1*10
1110
```
